# Playwright 102

Playwright end-to-end tests for the LambdaTest Selenium Playground.

## Setup

```bash
npm install
npx playwright install chromium
```

## Run tests

```bash
npx playwright test
```

## View report

```bash
npx playwright show-report
```

## HyperExecute

```bash
./hyperexecute --user YOUR_USERNAME --key YOUR_KEY --config hyperexecute.yaml
```
